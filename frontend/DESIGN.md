# Design

**Color Strategy:** Full palette. Data-heavy interface needs deliberate color use to distinguish between sections, parameters, and costs. Emphasize contrast for readability.
**Theme:** Light (default) with strong dark mode support for industrial/factory environments.
**Typography:** Monospaced or highly geometric sans-serif for numbers to ensure tabular data aligns perfectly. Strict hierarchy.
**Style:** Professional, geometric, full-width. Less padding around the main container to utilize the screen real estate for large tables. Clear grid lines, subtle borders, avoid soft drop shadows in favor of crisp 1px borders.
